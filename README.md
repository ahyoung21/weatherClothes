## 💰모임통장

### Getting Started

```
npm install
npm run dev
```

### 구현 내용

NextJS + TypeScript + recoil 로 작업
openweathermap api 를 활용
그날의 날씨에 따라 옷을 추천

### 개발 환경 및 버전

1.  Node.js 17.3.1
2.  NPM 8.3.0
3.  react 18.2.0
4.  react-dom 18.2.0
5.  react-router-dom 6.3.0
6.  next 13.0.2
7.  typescript 4.8.4
8.  styled-components 5.3.6
9.  axios 0.21.4
